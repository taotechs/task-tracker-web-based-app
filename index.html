Add a Task

How to add a task:

To add a new task, click on the "Add Task" button above. You will be redirected to the Add Task page where you can enter the details of your task and save it.

List Tasks

How to list tasks:

To view the list of tasks, click on the "List Task" button above. You will be redirected to the List Task page where you can see all the tasks you have added.
